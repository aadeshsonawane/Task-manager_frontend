# Task Manager App

A Task Management Web Application built with MERN Stack.

## Tech Stack
- MongoDB, Express.js, React.js, Node.js
- Bootstrap, JWT, Bcrypt, Axios

## Features
- User Register & Login
- Add, Edit, Delete Tasks
- Toggle Status (Pending → Incomplete → Completed)
- Search, Filter & Pagination
- Protected Routes

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## .env (Backend)
MONGO_URI=your_mongodb_uri
SECRET_KEY=your_secret_key
PORT=5002
